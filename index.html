증빙서류 취합 시스템 사용 안내

각 트랙 조교 선생님용 · 울산대학교 입학관리팀

1. 시스템 소개

진로체험 프로그램 운영 후 제출하는 교원참여보고서와 조교 업무일지를 웹페이지에서 직접 업로드하여 취합하는 시스템입니다. 이메일/카톡으로 보내실 필요 없이, 본인 트랙 페이지에 접속해 파일을 올려주시면 됩니다.

2. 접속 방법

접속 주소 (URL)
https://uou-career.uk

별도의 로그인이나 비밀번호가 필요하지 않습니다. 주소만 알면 바로 접속할 수 있으니 즐겨찾기에 추가해두시면 편합니다.

지원 브라우저 Chrome, Edge, Whale 등 최신 브라우저 권장. Internet Explorer 는 지원하지 않습니다.

3. 업로드 절차

  1. 본인 트랙(전공) 카드 클릭 홈 화면에 표시된 트랙 카드 중 본인이 담당하는 트랙을 클릭합니다.
  2. 해당 회차(고교/일자) 행에서 [열기] 클릭 본인 트랙이 진행되는 모든 회차가 미리 등록되어 있습니다. 해당 일자의 [열기] 버튼을 누릅니다.
  3. [교원참여보고서] 또는 [조교 업무일지] 영역에 파일 업로드 두 종류의 슬롯이 따로 있습니다. 각 슬롯에 해당 문서를 올려주세요.
  4. 업로더 이름 입력 (선택) 본인 이름을 적으면 누가 올렸는지 기록됩니다. 비워두셔도 무방합니다.
  5. 파일 선택 후 [업로드] 클릭 점선 영역을 클릭해 파일을 선택하거나, 파일을 영역으로 드래그해서 놓으면 됩니다. 한 번에 여러 파일 선택 가능합니다.

4. 수집하는 파일 종류

교원참여보고서

특강을 진행한 교수님께서 작성하시는 보고서. 트랙당/회차당 1개 파일 원칙. (PDF 또는 한글)

조교 업무일지

해당 회차를 보조한 조교 선생님의 업무 기록. 트랙당/회차당 1개 파일 원칙. (PDF 또는 한글)

파일 형식 / 크기 PDF, HWP, HWPX, DOCX, 이미지 모두 가능. 파일당 최대 50MB. 한 번에 여러 개 동시 업로드도 가능합니다.

5. 진행 상황 확인

홈 화면의 본인 트랙 카드에 진행률이 표시됩니다.

  • 교원참여보고서 X/Y — Y개 회차 중 X개 회차에 보고서가 업로드된 상태
  • 조교 업무일지 X/Y — 동일하게 업로드 진행 상황

세션 페이지(회차 상세)에서는 각 슬롯 옆에 업로드된 파일명이 표시됩니다.

6. 자주 묻는 질문

잘못된 파일을 업로드했습니다. 어떻게 지우나요?

조교 선생님은 직접 파일을 삭제하실 수 없습니다. 입학관리팀(관리자) 에게 연락 주시면 삭제해드립니다. 삭제 요청 시 어느 트랙·어느 회차·어느 파일인지 알려주세요.

파일을 한 번에 여러 개 올려도 되나요?

네, 가능합니다. 파일 선택 창에서 Ctrl 키를 누른 채 여러 파일을 클릭하거나, 여러 파일을 한꺼번에 드래그해서 점선 영역에 놓으시면 됩니다.

업로드된 파일을 확인하고 싶습니다. 다운로드 가능한가요?

조교 선생님 화면에서는 보안상 파일 다운로드가 비활성화되어 있습니다. 파일명만 확인 가능합니다. 내용 확인이 필요하시면 입학관리팀에 요청해주세요.

내 트랙이 목록에 없습니다.

홈 화면에는 5월 진로체험에 참여하는 트랙만 표시됩니다. 본인 트랙이 보이지 않으면 입학관리팀에 문의해주세요.

접속이 안 됩니다.

시스템은 운영 시간 중에만 접속 가능합니다. 오랫동안 접속이 안 되면 입학관리팀에 문의해주세요.
